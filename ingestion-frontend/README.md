# Niyanta - Knowledge Ingestion & Q&A

Clean, Claude AI-inspired frontend for knowledge ingestion and Q&A.

## Quick Start

```bash
npm install
npm run dev
```

Opens on `http://localhost:3000`

**Backend must be running on `http://localhost:8000`**

## Features

- Multi-source ingestion (GitHub, web, PDF, YouTube, Reddit, RSS)
- Q&A with cited sources
- Source filtering
- Auto-generated digests
- Real-time progress tracking
- Claude AI design language

## Tech Stack

- React 18 + Vite
- Tailwind CSS
- React Router
- React Markdown

## Build

```bash
npm run build
npm run preview
```
